Esse foi o projeto final do curso de Data Analytics da Ironhack. 
A pergunta que motivou esse projeto foi: por que alguns animais de extimação são adotados e outros não?
E existe algo que você pode fazer para melhorar a chance do seu pet ser adotado?

A primeira análise foi feita com os dados disponibilzados na competição do kaggle:kaggle.com/c/petfinder-adoption-prediction/data
Essa análise pode ser vista nos notebooks análise de dados, LGBM e Logistic Regression

Em seguida fiz um webscrapping do site amigonaosecompra.com.br para formar um banco de dados similar para o Brasil. 
Esse processo pode ser visto no notebook Scrapping e Limpeza de dados

De acordo com as análises as características mais importantes são descrições mais compridas e ter pelo menos 3 fotos.
